What is a website?

A website is a collection of information, images, and videos that are accessible on the internet/WWW(world wide web).
It is a collection of web pages, which are text documents written in HTML and CSS and become interactive with java script.

What is html/css/javascript?

HTML is the standard markup language for creating Web pages. CSS is the standard style sheet language for describing the presentation of Web pages, and JavaScript is a programming language that adds interactivity to HTML pages.
